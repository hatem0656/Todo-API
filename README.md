# Todo-API

API created to handle CRUD operations for a todo app

# Avaliable Endpoints

## Get all todos

### Request

`GET /api/todos/`

\_method=GET http://localhost:7000/thing/1

## Create a new todo

### Request

`POST /api/todos/`

\_method=POST http://localhost:7000/todos

## Delete a todo

### Request

`DELETE /api/todos/id`

\_method=DELETE http://localhost:7000/todos/2/
